# Project Companion Inbox

这里存放还没有进入 `40-Projects/` 的项目候选、项目卡点、下一步建议和阶段总结草稿。

## Use This For

- 一个想法可能变成项目，但还没有明确目标；
- Hermes / ChatGPT / Codex 发现某个学习任务应该转成 build action；
- 某个 active project 出现卡点，需要 Tony 判断；
- 周报里出现的项目推进建议，需要进一步确认。

## Do Not Use This For

- 正式项目状态页；
- canonical knowledge；
- 已经接受的 playbook；
- runtime log 或工具临时文件。

## Promotion Rule

只有满足以下条件，才从这里进入 `40-Projects/`：

```text
有明确目标
有当前状态
有下一步动作
Tony 或 Codex 明确判断值得推进
```
