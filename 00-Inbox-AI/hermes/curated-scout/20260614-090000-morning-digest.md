### 🔥🔥🔥🔥 主线 D: Agent 记忆研究大爆发 — 从"百花齐放"到"模式收敛"

在 6/12 上午版覆盖的 5 篇 Agent 记忆内容之上，6/12-6/14 出现了 **8 篇新 arXiv 论文**，涵盖从图谱到 KV-cache 原生的全频谱方法。这不是渐进积累——是领域正在寻找最优范式的关键阶段。

8. **MRAgent: 图谱记忆 + 主动重建** (arXiv 2606.06036): 将记忆组织为 Cue-Tag-Content 图谱，LLM 推理直接嵌入记忆检索过程。LoCoMo 基准 +23.3%（Gemini backbone），同时降低 token 和运行成本。
   [来源](https://arxiv.org/html/2606.06036v1)

9. **MemArt: KV-Cache 原生记忆** (OpenReview): 将 Agent 记忆直接存储在 LLM 的 KV Cache 中，通过潜在空间注意力分数检索。prefill tokens 减少 91-135×，准确率提升 11%。这是范式级别的创新——记忆从"明文检索"走向"模型原生"。
   [来源](https://openreview.net/pdf?id=YolJOZOGhI)

10. **Engram: 双时态知识图谱 + 8× 更少 tokens +10.4 分准确率** (arXiv 2606.09900): "少即是多"的极端案例——用 8× 更少的 tokens 实现了比完整上下文更高的准确率。开源，含评估工具。
    [来源](https://arxiv.org/html/2606.09900v1)

11. **其他重要论文**: AdMem（统一语义/情景/程序性记忆 + 多 Agent 架构）、Mage（执行状态管理 + 55% token 减少）、Rosetta Memory（跨 LLM 记忆适配）、CraniMem（神经认知启发门控记忆）、Infini Memory（主题文档记忆架构）。

**信号意义**: 6/12 上午版我们观察到 Agent 记忆"从百花齐放进入模式收敛"。本周的 8 篇论文证实了这一点——但收敛方向不是单一的。出现了两条清晰的竞争路线：**图谱/文档派**（MRAgent, Infini Memory, CraniMem）vs **KV-Cache 原生派**（MemArt）。后者是全新的范式，可能定义下一代 Agent 记忆架构。

**信号强度**: 🟢🟢🟢🟢 范式竞争级 | ✅ P1 in-progress: `agent-memory-architecture-package`

## 四、值得关注的跨故事主题

1. **AI 的"三线作战"**: 同一周末：美国政府强制下线 Anthropic 模型（**治理线**），SpaceX IPO 创历史纪录（**资本线**），Microsoft 推出自研模型挑战 OpenAI（**竞争线**）。三条线独立但相互强化——AI 正在从技术故事变为治理故事、资本故事和地缘政治故事。

2. **Agent 记忆的"范式之争"**: MemArt 的 KV-Cache 原生记忆 vs MRAgent/Infini Memory 的图谱/文档记忆，代表了两种根本不同的 Agent 架构哲学。前者将记忆嵌入模型内部（高效但耦合），后者保持记忆与模型的分离（灵活但效率低）。这将是 2026 下半年 Agent 架构的核心争论。

3. **Anthropic 悖论**: Anthropic 同时是 (a) 收入增长最快、最接近盈利的 AI 公司，(b) 因安全哲学与政府持续对抗的公司，(c) 第一个被政府强制下线模型的 AI 公司。这三重身份的张力将在 IPO 过程中被放大到极致。对 Tony 的多模型策略：Claude 模型的可用性风险管理现在成为一个需要明确应对的新维度。
