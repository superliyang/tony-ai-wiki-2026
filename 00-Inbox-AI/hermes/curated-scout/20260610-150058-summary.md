### 📌 建议关注

1.  **Agent 记忆系统**：本周有多篇文章从不同角度探讨了 Agent 记忆的缺陷与解决方案。这不再是“要不要记忆”的问题，而是“如何设计记忆”的问题。**记忆优先设计**和**记忆即证据**等新范式值得深入研究，它们可能是构建可靠、长期运行的 Agent 的关键。

2.  **MCP 协议生态的爆发**：从 Apify 的 MCP 连接器到 Claude 的可观测性工具，再到 Google 的 WebMCP 标准，MCP 协议正在从概念走向工程实践。它正在成为连接 AI Agent 与外部世界的“通用语言”。关注 MCP 生态的成熟度、标准化进程以及新的应用场景（如浏览器自动化）将非常有价值。

3.  **Agent 的可靠性保障**：形式化验证（Lean4Agent）的出现，为解决 Agent 的“幻觉”和“不可预测性”提供了数学层面的解决方案。虽然目前处于早期，但这代表了从“经验主义”向“严谨工程”的转变，对于金融、医疗等高风险领域的 Agent 应用至关重要。
