## 这个专题在解决什么问题

这个专题不是按 `GitHub stars` 做仓库收藏夹，而是要回答：

- 现在 AI 领域哪些开源项目值得深学
- 它们分别位于哪一层 stack
- 哪些项目代表一个模式，哪些只是壳层
- 哪些项目适合 `MacBook Pro Max M4` 本地实验
- 哪些项目适合迁移进工作体系
- 怎样把 repo 观察、模式抽象、watchlist 和实验计划接成长期控制塔

## 当前主线

- `LiteLLM`：模型网关与统一接入
- `Langfuse`：观测、评测、运维
- `LangGraph`：生产级编排
- `AutoGen`：多 Agent 协作范式
- `OpenHands`：Coding Agent 落地
- `browser-use`：网页动作面
- `MCP Servers`：标准化工具接入生态
- `A2A`：Agent 间互联
- `Claude Code 生态扩展`：skills、plugins、subagents、hooks、MCP、GitHub Actions 与安全/质量门禁的开源扩展样本
