# 开源 AI 栈组合样例：本地 Agent 与 Serving 研究台

## 目标

用一套最小组合来说明“开源项目不是单独学，而是按 stack 组合学”。

## 一个很实用的组合

- 本地运行：[[../03-Projects/Ollama]] 或 [[../03-Projects/MLX]]
- agent runtime：[[../03-Projects/LangGraph]]
- memory：[[../03-Projects/LangMem]]
- observability：[[../03-Projects/Langfuse]]
- regression gate：[[../03-Projects/Promptfoo]]
- self-improving / personal runtime：[[../03-Projects/OpenClaw]]

## 这类组合值得学什么

- 哪层是底座
- 哪层是壳层
- 哪层是治理与门禁
- 哪层可以本地跑，哪层更适合放云上
