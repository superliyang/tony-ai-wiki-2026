# claude-code-templates

## 它解决什么问题

`claude-code-templates` 提供 Claude Code 的 agents、commands、settings、hooks、MCPs、skills 和项目模板，并用 CLI 做浏览、安装、健康检查和监控。

## 为什么现在值得关注

它像 Claude Code 生态的“组件仓库 + 安装器 + 诊断工具”，很适合初学者快速看到生态全貌。

## 它在 stack 的哪一层

- 属于 `template catalog / installer / monitoring tool`
- 覆盖能力安装、配置管理、会话监控、插件 dashboard

## 核心组件与架构

- agents
- slash commands
- MCP integrations
- settings
- hooks
- skills
- analytics / chats / health-check / plugins dashboard

## 最适合它的场景

- 快速安装常用 Claude Code 组件
- 浏览社区模板
- 观察当前 Claude Code 配置健康度
- 学习组件如何组合成开发栈

## 和相邻项目怎么区分

- 和 [[anthropics-skills]]：它更偏集合和工具；Anthropic skills 更偏官方 skill 样例
- 和 [[SuperClaude Framework]]：它更像模板市场；SuperClaude 更像一套方法框架

## 风险与边界

- 组件来源多，需要逐个看 license 和权限
- 自动安装前应确认是否改 `.claude/`、settings、hooks 或 MCP 配置
