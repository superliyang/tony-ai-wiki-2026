# awesome-claude-skills

## 它解决什么问题

`awesome-claude-skills` 是 Claude Skills、resources 和 tools 的 curated list，帮助发现可复用 skill 设计样本。

## 为什么现在值得关注

当你想自定义 Claude Code 能力时，最难的不是写 Markdown，而是判断哪些任务值得沉淀成 skill、description 怎么写、资源如何组织。

## 它在 stack 的哪一层

- 属于 `skill discovery / skill pattern catalog`
- 位于 skill 自定义学习层

## 核心组件与架构

- skill repositories
- community resources
- examples and tools

## 最适合它的场景

- 找 skill 设计样本
- 学习 skill 命名、description 和结构
- 扩展个人或团队 skill library

## 和相邻项目怎么区分

- 和 [[anthropics-skills]]：官方 repo 更适合做规范基线；awesome list 更适合发现社区多样性
- 和 [[awesome-claude-plugins]]：它看 skill；后者看 plugin bundle

## 风险与边界

- skill 本身也可能包含脚本和外部资源，不应只看 README
- 需要把高风险 skill 和只读/解释类 skill 分开管理
