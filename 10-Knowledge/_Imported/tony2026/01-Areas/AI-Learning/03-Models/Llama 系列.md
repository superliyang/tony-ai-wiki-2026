# Llama 系列

## 简介

`Llama 系列` 是 [[Meta AI]] 最关键的模型家族之一，也是 open-weight 生态中的核心节点。

## 为什么重要

- Llama 是 open-weight 路线被全球开发者真正大规模采用的重要里程碑之一
- 它让“模型能力如何通过开放传播出去”这件事变得非常具体

## 你可以把这个模型家族当成什么来理解

- 可以把 Llama 看成闭源 API 路线之外的另一条主干
- 它的意义不仅在模型本身，也在于它如何放大了微调、部署、蒸馏和社区创新

## 学习时重点看什么

- 为什么 open-weight 会显著改变开发者生态
- Meta 为什么愿意押注这条路线
- Llama 和 GPT、Qwen、DeepSeek 在传播方式上的差别是什么
