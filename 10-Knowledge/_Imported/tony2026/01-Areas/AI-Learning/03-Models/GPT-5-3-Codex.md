# GPT-5.3-Codex

## 简介

`GPT-5.3-Codex` 是 OpenAI 在 `2026-02-05` 发布的 agentic coding 旗舰，也是截至 `2026-04-07` 最值得单独记住的 coding 模型之一。

## 为什么重要

- OpenAI 明确把它定位成“最强的 agentic coding model”
- 它不只提升 coding，本身也在往 research、tool use、complex execution 和 professional work 扩张
- 它比 `GPT-5.2-Codex` 更快，同时把 `SWE-Bench Pro`、`Terminal-Bench`、`OSWorld` 和 `GDPval` 这些能力收成了一个更通用的工作模型

## 这条路线代表什么

- 它代表 `coding model -> general collaborator on the computer`
- OpenAI 在这里传达的不是“再做一个写代码模型”，而是 coding agent 已经开始向通用工作代理扩张

## 学习时重点看什么

- 为什么 coding model 会自然接到 `computer use`
- 为什么 agentic coding 的下一步不是更多补全，而是更长任务、更强反馈循环、更强人机协同
- 为什么 OpenAI 会在这条线上同时做出 `GPT-5.3-Codex` 和 `GPT-5.3-Codex-Spark`

## 你学习这个模型时最该建立的判断

- coding 已经不是应用层附属，而是 frontier 模型分层本身
- `GPT-5.3-Codex` 让 `Codex` 从代码 agent 更明显地走向通用 computer-based collaborator
