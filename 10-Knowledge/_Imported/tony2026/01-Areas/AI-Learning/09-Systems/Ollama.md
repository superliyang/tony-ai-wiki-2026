# Ollama

## 它是什么

`Ollama` 是把本地运行模型这件事做得极简化的系统。

## 为什么它适合学习

因为它能让你很快进入：

- local model serving
- prompt experimentation
- local agent / RAG app prototyping

而不用先把底层 runtime 细节全啃完。

## 它最适合什么

- 本地模型试跑
- 本地 API 服务
- 快速验证 RAG / agent / tool workflow

## 它不替代什么

- 不替代对底层推理机制的理解
- 不替代 `llama.cpp` / `MLX-LM` 这种更底层的学习价值
