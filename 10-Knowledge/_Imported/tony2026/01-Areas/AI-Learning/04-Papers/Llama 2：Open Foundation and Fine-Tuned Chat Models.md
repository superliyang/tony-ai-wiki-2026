# Llama 2：Open Foundation and Fine-Tuned Chat Models

## 简介

`Llama 2` 是 open-weight 路线真正进入全球开发者主线的一篇代表论文 / 报告。

## 核心想法

它不只是发布一组模型，而是把 `open foundation model + chat finetuning + 商业可用叙事` 一起推出去。

## 为什么重要

- 它让 open-weight 路线从研究社区扩散到更广泛的产品和创业场景
- 它直接推动了本地部署、私有部署和行业微调生态
- 它让很多团队第一次觉得“不一定只能走闭源 API 路线”
