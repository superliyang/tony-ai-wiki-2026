# 项目总览

## 这个项目包是干什么的

`Memory Lab` 不是让你一上来造一个完美 memory system，而是把最容易说空的几层：

- state vs memory
- artifact separation
- hot-path write vs background consolidation
- semantic / episodic / procedural memory
- pollution / staleness / forgetting
- self-maintenance vs controlled self-improvement
- learnings、promotion、skill extraction 和 release gate
- poisoning、shared memory boundary 和 rollback discipline
- incident-fed eval、shadow rollout 和 plugin wrapper
- local marketplace、hook payload debugging、incident replay pack 和 candidate-skill rollout gate

压成几个可做的小实验。
