# AI Coding Lab

## 项目定位

`AI Coding Lab` 是把 [[../../../AI-Learning/06-Topics/AI Coding 专家能力体系|AI Coding 专家能力体系]] 从“知识图谱”推进到“可练习、可评估、可复盘”的实战项目包。

它不追求一次性搭一个大平台，而是用一组小而真实的 repo 任务，训练并验证：

- 任务建模
- 上下文工程
- 工具面设计
- skills / subagents / hooks / MCP / plugins 沉淀
- 测试、review、审计和安全门禁
- 团队试点与复盘机制

## 目标

这个 Lab 要回答：

- 我是否真的会用 coding agent 完成真实 repo 任务
- 我能否把一次成功经验沉淀成可复用能力
- 我能否识别哪些权限、命令和外部工具应该被限制
- 我能否用评测包证明 AI Coding 质量，而不是只凭体感
- 我能否把个人经验升级为团队工作流

## 实验批次

### Batch 1：个人闭环

- bugfix
- 小型 refactor
- 测试补全
- PR summary
- review follow-up

目标：证明 agent 能完成真实 repo 的低风险任务。

### Batch 2：能力沉淀

- 抽一个 `repo-map skill`
- 抽一个 `test-writer skill`
- 抽一个 `code-reviewer subagent`
- 加一个 `post-edit test reminder hook`

目标：从一次性使用变成可复用能力。

### Batch 3：团队试点

- 选择 1 个低风险 repo
- 定义 `CLAUDE.md / AGENTS.md`
- 定义权限、敏感路径、验证命令
- 使用评测矩阵做首轮验收

目标：形成团队最小可试点机制。
