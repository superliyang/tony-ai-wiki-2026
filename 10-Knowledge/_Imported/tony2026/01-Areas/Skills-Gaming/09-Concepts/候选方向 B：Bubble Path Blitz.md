# 候选方向 B：Bubble Path Blitz

## 一句话

在短时间里通过角度、连锁和路径规划拿高分的 bubble/pathing 竞赛玩法。

## 为什么值得考虑

- 市场教育成本低
- 视觉反馈强
- skill 和 score 的关系比较直观
- 很适合 event 化和 modifier 设计

## 主要风险

- 需要把随机性压低
- 需要让高分路径更可解释
- 容易和现有 bubble game 太相似

## 我的判断

这是一个很稳的第二梯队候选，适合当 fallback 或双线探索方向。
