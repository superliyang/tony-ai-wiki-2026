# Unity AI 与 AI Gateway

## 这套东西在行业里代表什么

它代表的不是“再加一个生成图片按钮”，而是：

- AI 正被正式接进 editor workflow
- 第三方 agent 开始被安全地接进 Unity
- AI 不只是内容生成，而是工作流加速器

## 官方最值得注意的信号

### `Unity AI`

Unity 已经把它定义成：

- integrated directly into the Unity Editor
- 基于第三方 generative models
- 面向 editor 内工作流

### `Unity AI Gateway`

Unity 在 `2025-11-19` 的 `Unite 2025` 里明确提出：

- coming in 2026
- securely connect third-party AI agents with Unity
- deep context on scene, hierarchy, assets, platform targets

这意味着：

- 以后 editor agent 不是外挂，而会逐步变成平台能力

## 对你们这类 hackathon 团队的意义

如果最终主战场是 `Unity`，它最有价值的是：

- editor 内上下文更强
- 工具链更统一
- 第三方 agent 接入更自然

## 当前阶段的现实判断

- 它更像“创作工作流和 editor 协作层”
- 不直接替代你们的 gameplay 设计与 fairness 设计
