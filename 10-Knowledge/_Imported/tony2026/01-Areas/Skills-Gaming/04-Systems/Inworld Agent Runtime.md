# Inworld Agent Runtime

## 为什么它代表前沿

`Inworld` 不只是聊天 NPC，而是在往：

- Agent Runtime
- Unreal runtime integration
- reactive + reasoning 架构

推进。

## 官方最值得看的点

- docs 直接写 `Create and chat with an AI character with Agent Runtime`
- Unreal AI Runtime SDK 强调更快地把 AI-powered games 做出来
- 它的架构已经开始支持不仅是对话，还包括更长程的 decision / director 式逻辑

## 对 hackathon 的现实意义

适合学它：

- agent runtime 的产品形态
- AI NPC 不只是 reply，而是 scene-aware behavior

但不建议你们在 `skills game V1` 里把主要精力花在复杂 NPC 上。
