# NVIDIA ACE for Games

## 它是什么

`NVIDIA ACE` 不是单一模型，而是一套给 middleware 和 game developers 的角色 AI 技术栈。

官方定义里最关键的是：

- knowledgeable
- actionable
- conversational in-game characters

## 为什么它重要

它代表的是“AI NPC / companion / teammate”正在从 demo 走向可集成产品栈。

## 对游戏产品的意义

最适合的场景：

- companion
- tutorial guide
- social NPC
- narrative support character

## 对 skills gaming 的现实判断

对你们当前的 `V1`，它更适合：

- onboarding guide
- announcer / host
- optional companion-like shell

不适合让它介入：

- 核心公平性
- 核心比赛判定
- 核心 score 机制
