# Google AI Studio 内容与素材生产包

## 内容侧任务

### Gemini

用于：

- game brief
- tutorial copy
- result copy
- qualifier copy
- pitch one-pager
- 3 分钟 demo narration

### Imagen

用于：

- UI moodboard
- icon directions
- promotional art
- event banner ideas

### TTS

用于：

- tutorial host voice
- match result announcer
- trailer voiceover

### Veo

用于：

- 短 trailer
- mood clip
- 宣传视频草稿

## 使用原则

- 先定 art direction，再批量生成
- 先定玩法与术语，再批量生成文案
- 所有对外素材都要人工统一语气和风格

## 最值得优先生成的东西

1. tutorial 文案
2. result 文案
3. pitch one-pager
4. trailer 旁白
5. UI moodboard
