# 网络安全

## 定位

网络安全关注系统之间的连接、边界、路径和流量控制。

它不是“防火墙配置”这么窄，而是回答：

- 谁能连进来
- 流量走哪条路
- 哪些协议和端口应该存在
- 异常流量如何被发现
- 攻击发生时如何隔离和止血

## 关键问题

- 外部入口在哪里：公网 IP、域名、API gateway、VPN、CDN、WAF
- 内部横向移动如何限制：segmentation、zero trust、least privilege
- DDoS、扫描、爆破、DNS 劫持、流量劫持如何发现
- 网络日志如何接入检测和响应

## 主要控制

- network segmentation
- firewall / security group
- WAF
- DDoS protection
- IDS / IPS
- VPN / ZTNA
- DNS security
- traffic monitoring
