# 客户端安全

## 定位

客户端安全关注用户、员工和设备侧的攻击面。

它包括：

- PC / Mac
- 手机 App
- 浏览器
- 企业终端
- 客户端 SDK
- IoT / 边缘设备

## 关键问题

- 设备是否可信
- 客户端是否被篡改、调试、Hook、逆向
- 用户是否被钓鱼、恶意软件、浏览器插件攻击
- 客户端存储了哪些敏感数据
- 客户端和服务端之间如何认证、加密和防重放

## 主要控制

- endpoint protection
- EDR / XDR
- MDM / device posture
- app hardening
- certificate pinning
- secure local storage
- anti-tamper / anti-debug
- phishing protection

## 和其他分支的关系

- 和 [[身份与访问安全]] 连接 MFA、device trust、conditional access
- 和 [[应用安全与 API 安全]] 连接移动端 API 调用和业务逻辑保护
- 和 [[数据安全与隐私保护]] 连接本地缓存、日志和敏感数据
- 和 [[安全运营、检测与响应]] 连接 EDR 告警和终端取证
