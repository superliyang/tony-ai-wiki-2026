# 安全运营、检测与响应

## 定位

安全运营关注攻击发生时能不能看见、判断、响应、恢复和复盘。

它不是“买一个 SIEM”这么简单，而是：

`日志 -> 检测 -> 告警 -> 分诊 -> 响应 -> 取证 -> 恢复 -> 复盘 -> 改进`

## 关键问题

- 哪些日志必须采集
- 什么行为算异常
- 告警如何降噪和分级
- 谁来响应，多久响应
- 如何止血、隔离、取证、恢复
- 事故如何反哺规则、控制和流程

## 主要控制

- SIEM
- SOAR
- EDR / XDR
- threat intelligence
- detection engineering
- incident response
- threat hunting
- DFIR
- tabletop exercise
- post-incident review
