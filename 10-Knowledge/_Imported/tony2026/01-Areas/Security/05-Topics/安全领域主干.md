# 安全领域主干

## 一句话

计算机安全是一套围绕 `资产、攻击面、威胁、控制、检测、响应、恢复、治理和合规证明` 的系统工程。

## 上帝视角：安全到底在保护什么

安全不是抽象地“防黑客”，而是在保护这些东西：

- `身份`：用户、员工、服务账号、机器身份、API key、证书
- `终端`：PC、手机、浏览器、客户端、IoT、办公设备
- `网络`：内网、外网、云网络、边界、DNS、流量路径
- `应用`：Web、移动端、API、微服务、业务逻辑
- `数据`：个人信息、支付数据、交易记录、商业机密、日志、模型数据
- `基础设施`：主机、容器、Kubernetes、云账号、CI/CD、存储
- `供应链`：依赖、镜像、构建系统、第三方 SDK、外包服务
- `业务流程`：审批、支付、客服、运维、发布、权限变更
- `合规证据`：审计日志、控制证明、风险评估、数据处理记录

## 安全的九个问题

1. `资产是什么`：我们到底要保护什么？
2. `攻击面在哪里`：攻击者能从哪里进来？
3. `威胁是谁`：脚本小子、犯罪团伙、内部人、供应链攻击者、国家级攻击者？
4. `弱点在哪里`：漏洞、错误配置、弱身份、流程缺口、人员误操作？
5. `控制怎么设计`：预防、隔离、最小权限、加密、校验、审批、审计？
6. `怎么发现异常`：日志、告警、行为分析、威胁情报、检测规则？
7. `怎么响应恢复`：止血、隔离、取证、修复、通报、复盘？
8. `怎么证明合规`：控制证据、审计轨迹、地区要求、行业标准？
9. `怎么持续改进`：把事故、红队、演练、审计发现变成机制？

## 十条主干

### 1. [[身份与访问安全]]

核心问题：谁可以访问什么，凭什么，访问后能做什么。

典型关键词：

- IAM
- MFA
- SSO
- OAuth / OIDC
- RBAC / ABAC
- PAM
- service account
- secret management

### 2. [[网络安全]]

核心问题：流量如何进入、穿过和离开系统，哪些路径应该被允许。

典型关键词：

- firewall
- IDS / IPS
- VPN / ZTNA
- DNS security
- segmentation
- DDoS
- WAF
- network detection

### 3. [[客户端安全]]

核心问题：用户和员工手里的设备、浏览器、App、客户端如何成为攻击入口。

典型关键词：

- endpoint security
- EDR
- mobile security
- browser security
- malware
- phishing
- app hardening
- device posture

### 4. [[应用安全与 API 安全]]

核心问题：业务系统和 API 如何避免被越权、注入、滥用和逻辑攻击。

典型关键词：

- OWASP Top 10
- secure coding
- API security
- authz
- input validation
- SSRF
- SQL injection
- business logic abuse

### 5. [[数据安全与隐私保护]]

核心问题：敏感数据如何分类、最小化、加密、授权、审计和删除。

典型关键词：

- data classification
- encryption
- tokenization
- DLP
- privacy by design
- retention
- data residency
- consent

### 6. [[云原生与基础设施安全]]

核心问题：云账号、主机、容器、Kubernetes、网络和配置如何安全运行。

典型关键词：

- CSPM
- CNAPP
- container security
- Kubernetes security
- IaC scanning
- runtime security
- cloud IAM
- workload identity

### 7. [[供应链安全]]

核心问题：代码、依赖、构建、镜像、发布和第三方组件如何不被投毒。

典型关键词：

- SBOM
- SCA
- provenance
- signing
- CI/CD security
- dependency confusion
- artifact registry
- secure build

### 8. [[安全运营、检测与响应]]

核心问题：攻击发生时如何发现、判断、止血、取证、恢复和复盘。

典型关键词：

- SOC
- SIEM
- SOAR
- EDR / XDR
- incident response
- threat hunting
- DFIR
- playbook

### 9. [[安全治理、风险与合规]]

核心问题：安全如何变成组织机制，而不是安全团队的孤岛。

典型关键词：

- GRC
- risk register
- policy
- control
- audit
- third-party risk
- security awareness
- board reporting

### 10. [[地区合规与监管坐标]]

核心问题：不同地区和行业对数据、隐私、网络、支付、跨境传输和事件通报有什么要求。

典型关键词：

- GDPR
- CCPA / CPRA
- PIPL
- PCI DSS
- HIPAA
- SOC 2
- ISO 27001
- NIS2
- data residency

## 怎么学习才不碎

不要按工具学：

- 今天学 WAF
- 明天学 EDR
- 后天学 SIEM

要按问题学：

- 哪个资产被保护
- 哪个攻击面暴露
- 哪个威胁最现实
- 哪个控制最有效
- 哪个指标证明有效
- 哪个证据能被审计
