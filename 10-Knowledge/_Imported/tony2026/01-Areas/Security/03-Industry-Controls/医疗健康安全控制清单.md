# 医疗健康安全控制清单

> 医疗健康安全的核心是：保护 PHI/健康数据、医疗业务连续性、临床系统可用性、第三方共享和监管证据。

## 保护对象

- 患者身份、病历、诊断、处方、影像、检验、保险与支付数据。
- EHR/EMR、预约、药房、实验室、影像系统、远程医疗系统。
- 医生、护士、客服、运营、保险/合作机构访问权限。
- 医疗设备、IoMT、工作站、终端和网络分区。
- 数据共享、研究数据、脱敏数据、第三方接口和供应商。

## 主要威胁

- 勒索软件导致医疗服务中断。
- PHI 泄露、内部越权查看、批量导出。
- 账号接管、共享账号、离职账号未关闭。
- 医疗设备弱口令、老旧系统、补丁困难。
- 第三方供应商、实验室、保险接口泄露。
- 远程医疗 API、移动端和患者门户越权。

## 控制清单

| 控制域 | 必做控制 | 证据 |
|---|---|---|
| 身份与访问 | MFA、最小权限、紧急访问、离职关闭、访问审计 | 权限矩阵、访问日志、审计抽样 |
| PHI 数据保护 | 分类分级、加密、脱敏、最小化、导出审批 | 数据地图、DLP 报告、审批记录 |
| 业务连续性 | 备份、恢复演练、勒索隔离、关键系统 RTO/RPO | 备份报告、演练记录、恢复测试 |
| 终端与设备 | EDR、补丁、设备分区、老旧系统补偿控制 | 终端覆盖率、漏洞例外、网段策略 |
| 应用/API | 患者门户、远程医疗、第三方接口安全评审 | API 评审、测试报告、修复记录 |
| 检测响应 | PHI 异常访问、批量导出、勒索行为检测 | SIEM 规则、case、事件响应记录 |
| 供应商 | BAA/合同控制、数据共享评估、供应商安全审查 | 供应商评估、合同条款、证据包 |
| 合规证据 | HIPAA 安全控制、风险评估、策略确认 | 风险登记、控制映射、培训记录 |

## 平台组合

- IAM / PAM：医护、运营、供应商和紧急访问治理。
- DLP / DSPM：PHI 发现、分类、导出监控、异常访问。
- EDR / XDR：勒索软件、终端行为、医疗设备旁路监控。
- SIEM / SOAR：PHI 访问、批量导出、身份异常、勒索告警。
- GRC：HIPAA 控制、风险评估、供应商和培训证据。

## 关键指标

- PHI 数据资产覆盖率。
- 高敏数据访问审计覆盖率。
- 备份恢复演练成功率。
- 勒索关键检测覆盖率。
- 离职账号关闭时间。
- 供应商安全评估覆盖率。

## 常见陷阱

- 只关注合规文件，不验证备份和恢复。
- 医疗设备和老旧系统无法打补丁，却没有隔离和补偿控制。
- 紧急访问开了后没有审计和复盘。
- 内部越权查看 PHI 被忽略，只关注外部攻击。
