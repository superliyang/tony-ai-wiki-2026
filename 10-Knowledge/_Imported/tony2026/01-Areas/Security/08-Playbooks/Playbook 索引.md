# Security Playbook 索引

> Playbook 层的目标是：把安全知识从“知道概念”变成“遇到问题能做判断、能落地、能复盘”。

## 当前 Playbooks

- [[./安全快速诊断 Playbook|安全快速诊断 Playbook]]：遇到一个安全问题时，先判断资产、攻击面、威胁、影响和优先级。
- [[./企业安全架构落地 Playbook|企业安全架构落地 Playbook]]：从组织视角建立身份、数据、应用、云、供应链、运营和治理能力。
- [[./应用与 API 安全评审 Playbook|应用与 API 安全评审 Playbook]]：用于新系统上线、API 开放、重大变更和安全评审。
- [[./安全事件响应 Playbook|安全事件响应 Playbook]]：用于告警、入侵、泄露、勒索、云密钥泄露等事件响应。
- [[./数据安全与隐私评审 Playbook|数据安全与隐私评审 Playbook]]：用于敏感数据、个人信息、跨境、保留删除和审计证据。
- [[./安全学习与实战路线 Playbook|安全学习与实战路线 Playbook]]：用于明确学习阶段、实战练习、方向分支和掌握标准。
- [[./零信任落地 Playbook|零信任落地 Playbook]]：用于把身份、设备、应用、数据、网络和持续监控串成零信任路线。
- [[./漏洞管理与攻击面管理 Playbook|漏洞管理与攻击面管理 Playbook]]：用于资产发现、暴露面识别、漏洞排序、SLA、例外和验证。
- [[./供应链安全评审 Playbook|供应链安全评审 Playbook]]：用于评审代码、依赖、构建、产物、发布、第三方 SDK 和供应商。
- [[./云原生安全基线 Playbook|云原生安全基线 Playbook]]：用于云账号、IAM、Kubernetes、容器、IaC、CI/CD 和运行时基线。
- [[./SOC 检测工程 Playbook|SOC 检测工程 Playbook]]：用于按攻击场景设计日志、检测规则、分诊上下文和响应动作。
- [[./安全实战实验室路线 Playbook|安全实战实验室路线 Playbook]]：用于把资产盘点、威胁建模、安全评审、检测响应和 GRC 变成练习产物。
- [[./红队蓝队紫队演练路径 Playbook|红队蓝队紫队演练路径 Playbook]]：用于在授权范围内把攻击链、检测工程、事件响应和复盘改进串成演练闭环。
- [[./安全工具平台选型与落地 Playbook|安全工具平台选型与落地 Playbook]]：用于选择、PoC、接入和运营 IAM、AppSec、CNAPP、SIEM、DLP、GRC 等平台。
- [[./手机游戏端到端安全架构落地 Playbook|手机游戏端到端安全架构落地 Playbook]]：用于把手机游戏撞库、客户端、反作弊、经济、支付、运营后台、数据、供应链和 SecOps 串成可执行的全局安全架构。
- [[./手机游戏端到端安全环节治理手册|手机游戏端到端安全环节治理手册]]：用于逐环节梳理获客注册、登录会话、找回换绑、客户端、API、战斗、结算、经济、支付、客服、数据、供应链和安全运营的风险、动作、步骤和演进。
- [[./手游撞库与账号安全治理 Playbook|手游撞库与账号安全治理 Playbook]]：用于手机游戏被撞库后的止血、排查、账号安全加固和长期风控建设。
- [[./手游账号安全检测规则库|手游账号安全检测规则库]]：用于把撞库、账号接管、资产搬运、客服社工、退款套利等场景转成检测规则、处置动作和误伤复核。
