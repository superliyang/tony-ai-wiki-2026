# awesome-claude-plugins

## 它解决什么问题

`awesome-claude-plugins` 是 Claude plugins 的 curated list，用来发现通过 plugin system 扩展 Claude Code 的 commands、agents、hooks 和 MCP servers。

## 为什么现在值得关注

第三方插件增长很快，需要一个 discovery list 帮助初筛，但它更适合“发现线索”，不适合直接当可信安装源。

## 它在 stack 的哪一层

- 属于 `plugin discovery / curated list`
- 不等同于官方 marketplace

## 核心组件与架构

- plugin links
- 分类目录
- 简介与外部入口

## 最适合它的场景

- 找 Claude Code 插件线索
- 对比第三方插件覆盖面
- 为自己的 team marketplace 做参考

## 和相邻项目怎么区分

- 和 [[claude-plugins-official]]：官方目录更适合安装起点；awesome list 更适合发现社区项目
- 和 [[awesome-claude-skills]]：它看 plugin，后者看 skill

## 风险与边界

- curated list 不是安全认证
- 真正安装前必须回到每个 repo 的源码、权限、hooks、MCP 配置逐项审查
