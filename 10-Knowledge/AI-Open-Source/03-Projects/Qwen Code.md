# Qwen Code

Qwen Code 代表 open coding agent 与 Qwen3-Coder 模型协同的路线。它适合放在 AI Open Source 的 Watchlist 中持续跟踪，因为它不只是一个 CLI 工具，也反映中国开源模型生态进入 coding agent 工作流的方式。

## 为什么进 Watchlist

| Reason | 说明 |
|---|---|
| Coding Agent | 与 Codex / Claude Code / Cursor 等工作流形成对照 |
| Open Model Toolchain | 连接 Qwen3-Coder、open-weight code model、本地/云端执行 |
| China AI Ecosystem | 是观察 Qwen 生态工程化能力的重要入口 |
| Repo-level Work | 如果能稳定处理 repo 级任务，会影响 agentic coding 模型选择 |

## 需要跟踪的问题

- Qwen Code 的实际 workflow 和 Codex / Claude Code 有什么差异？
- Qwen3-Coder 在 repo 级任务、工具调用、长上下文上表现如何？
- 是否适合进入 Tony 的 coding agent 对比表？
- 是否能和本地 serving / vLLM / Ollama / LiteLLM 形成组合？
