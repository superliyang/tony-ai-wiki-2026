# Unsloth

Unsloth 是本地训练和 fine-tuning 方向值得长期跟踪的开源项目。它的价值不只是“训练更快”，而是代表个人和小团队把模型微调、RL、数据处理和本地实验工作台产品化的方向。

## 为什么进 Watchlist

| Reason | 说明 |
|---|---|
| Local Fine-tuning | 影响 Mac / 单机 / 小 GPU 环境下的实验路径 |
| Training Efficiency | 对学习 LoRA、SFT、RL、GRPO 等训练流程有工程参考价值 |
| Studio / Workflow | 如果 UI / Studio 化成熟，会降低本地实验门槛 |
| Model Coverage | 支持模型变化快，需要周期性跟踪 |
| License / Commercial Boundary | 本地工具链采用前要看 license 和使用边界 |
