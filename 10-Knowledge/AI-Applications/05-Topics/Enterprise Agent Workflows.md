# Enterprise Agent Workflows

## 这个主题是什么

`Enterprise Agent Workflows` 关注 agent 如何嵌入企业内部流程，连接知识、审批、系统和任务执行，而不是只做个人助手或聊天机器人。

## 为什么重要

- 企业真正关心的不只是“模型强不强”，而是：
  - 能不能进入现有流程
  - 能不能接企业系统
  - 能不能控制权限、审计和责任
- 这类场景一旦跑通，往往比单一聊天产品有更高的组织价值

## 常见场景

- 内部知识问答与流程导航
- 跨系统任务编排
- 工单处理与初步归因
- 审批辅助与信息准备
- 例行报告、巡检和状态同步

## 真正难的地方

- 企业流程比个人工作更强调权限、审计和责任边界
- 工具接得越多，治理难度越高
- 企业 agent 不只是模型系统，也是组织流程系统

## 推荐工作方式

- 先做窄场景、固定流程、可审计任务
- 先连接低风险系统，再碰高风险写操作
- 所有企业 agent 都要和审批、日志、回滚、权限模型一起设计
- 不要一开始就追求“全能企业 agent”
