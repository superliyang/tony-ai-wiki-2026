# OpenClaw 产品卡

## 简介

`OpenClaw` 更像一个 self-hosted personal agent runtime。它的重点不是“最强模型接口”，而是一个带 gateway、workspace、memory、hooks、heartbeat、cron 的长期运行 agent 系统。

## 核心功能

- gateway/control plane
- session 与 memory 管理
- typed tools
- hooks、heartbeat、cron 驱动长期运行
- 多渠道 assistant 入口

## 价值点

- 非常适合研究“个人 agent runtime 应该怎么设计”
- 对想自托管、想长期运行、想实验 agent 自动化的人很有吸引力
- 它天然把 runtime、memory、automation 这些 agent 深水区问题放到桌面上

## 使用场景

- 个人研究助理
- personal assistant workflow
- self-hosted automation assistant
- 研究 agent runtime 架构的案例系统

## 边界与注意点

- 它不是现成面向大众的一站式 SaaS 产品
- 真正价值偏“可构建性”和“可持续 agent runtime”，不是即开即用的 consumer assistant
- 更适合作为系统案例和 runtime 研究对象

## 竞争与差异

- 和 `ChatGPT Agent` 比，它更偏 self-hosted runtime
- 和 `Claude Code`、`Cursor` 比，它不是以 coding workflow 为中心
