# 最终 shortlist：推荐方向与原因

## 一号候选：`Word Sprint Duel`

### 为什么它最适合现在

- UI-heavy，资产成本最低
- scoring 和 skill 很容易讲清楚
- AI 非常适合帮它加速词库、文案、UI、QA
- 对你这种偏平台与系统的人来说，最容易把精力放在 fairness、match flow、event logging 上

### 它的主要风险

- 词库质量和本地化
- 很容易做得太像普通 word puzzle

### 为什么风险可控

- 可以用非常小的词库先做 demo
- 可以把重点放在 duel framing 和 qualifier progression 上

## 二号候选：`Bubble Path Blitz`

### 为什么它值得保留

- 熟悉度高
- 视觉反馈强
- LiveOps / event modifier 空间大

### 为什么它是二号

- 需要更仔细地控制随机性
- 和现有 portfolio 容易相似，pitch 新鲜度稍弱

## 暂不推荐作为一号的方向

### `Precision Shot Arena`

风险主要在：

- 手感调参
- 客户端细节耗时
- demo 成败更依赖 execution quality

### `Tile Merge Relay`

风险主要在：

- 差异化不足
- 容易显得像保底方案而不是亮点方案

## 当前建议

- 主做：`Word Sprint Duel`
- 备份：`Bubble Path Blitz`
