# LiveOps 节奏与运营机制

## 这页在补什么

前面我们讲了 retention 和 rewards，这页讲的是：

- 这些东西如何组织成节奏
- 团队如何周周跑、月月跑

## 从官方动作里最值得学的 3 种节奏

### 1. 日常节奏

- daily freeplay
- daily challenge
- qualifier reset

`WorldWinner` 和 `Papaya` 的 daily tournament / qualifier 设计都说明：

- 每天都要给玩家一个简单、可重复进入的理由

### 2. 中周期节奏

- weekly event
- limited-time modifier
- featured mode rotation

### 3. 大事件节奏

- championship
- live finals
- seasonal ladder

## 对 hackathon 的建议

pitch 里至少要讲清：

- daily rhythm
- weekly event idea
- one bigger narrative event
