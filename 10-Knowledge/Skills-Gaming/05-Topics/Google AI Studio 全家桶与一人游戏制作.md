# Google AI Studio 全家桶与一人游戏制作

## 为什么我会优先推荐它

因为对一人或极小团队来说，它是少见的“从想法、图像、语音、视频到 agentic prototype”都能覆盖的组合。

## 最实用的 5 个使用位点

### 1. 创意与系统文案

- `Gemini`
- long context
- 竞品比较
- game brief
- onboarding copy

### 2. 美术探索

- `Imagen`
- moodboard
- UI icon 和 banner 方向
- card / tile / word-cell variants

### 3. 演示视频

- `Veo`
- pitch trailer
- store-like short video
- mood clip

### 4. 语音层

- `TTS`
- tutorial host
- announcer
- promo voice

### 5. 实时互动原型

- `Live API`
- 语音 guide
- 简单主持人
- interactive assistant demo

## 对 `Word Sprint Duel V1` 的最推荐组合

- `Gemini`：规则解释、文案、UI copy、任务拆分
- `Imagen`：UI direction、活动海报、素材变体
- `TTS`：教程播报、结算 announcer
- `Veo`：demo trailer

## 不建议过度依赖的地方

- 实时生成核心词库规则
- 实时决定比赛结果
- 让 AI 直接定义核心公平性
