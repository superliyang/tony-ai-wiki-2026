# 推荐方向：Word Sprint Duel V1

## 一句话

一个围绕“快速识别 + 路径选择 + 短局对战”设计的 word-based skills game 原型。

## 为什么它最适合现在

- 最容易做出清楚的 score pipeline
- 最容易解释 fairness
- 最容易做出完整 demo path
- 最能发挥 AI 在文案、UI、词库、QA 上的提效能力

## V1 只做什么

- 单一 core loop
- 单一 duel mode
- score submission
- result screen
- simple daily qualifier framing
- event logging

## V1 不做什么

- 真钱收付
- 大型运营后台
- 复杂 meta economy
- 多模式扩展
