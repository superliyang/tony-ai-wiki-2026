# 技能游戏技术架构作战手册

## 目标

做一个：

- 可玩
- 可提交 score
- 可讲 fairness
- 可解释系统边界

的技术 demo，而不是一套臃肿平台。

## 最小架构

### 必做

- game loop
- score calculation
- result / submission flow
- tournament shell
- event logging

### 可做

- simple leaderboard
- simple anomaly check
- eligibility placeholder
- reward placeholder

### 不做

- full wallet system
- full anti-cheat platform
- large-scale operations backend
- complete liveops console

## 技术评审最该讲什么

- score 是怎么生成的
- fairness 是怎么保证的
- 哪些地方未来会扩成平台
- 为什么这版故意不做更重的系统
