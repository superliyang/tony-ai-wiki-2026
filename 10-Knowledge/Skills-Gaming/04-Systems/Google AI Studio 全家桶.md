# Google AI Studio 全家桶

## 它为什么值得进这个专题

因为它不是单点模型，而是一套非常适合一人团队做原型的组合：

- `Gemini`：创意、文案、代码、长上下文分析
- `Imagen`：2D 视觉探索和素材变体
- `Veo`：短视频、trailer、演示素材
- `TTS`：主持人、旁白、NPC 声音、教学语音
- `Live API`：实时语音交互和更 agentic 的体验原型

## 对游戏创新马拉松最有价值的地方

### 1. 从想法到 brief

`Gemini` 很适合：

- 玩法发散
- 竞品比较
- 文案统一
- UI copy
- tutorial / result screen 文案

### 2. 从 moodboard 到界面素材

`Imagen` 很适合：

- 风格板
- UI icon 方向
- 卡片、棋盘、按钮变体
- 宣传图初稿

### 3. 从 demo 到宣传视频

`Veo` 很适合：

- pitch trailer
- 开场 mood clip
- 宣传视频草稿

### 4. 从文字到声音

`TTS` 和 `Live API` 很适合：

- 引导语音
- 结算播报
- 旁白
- 低门槛语音 NPC 原型

## 它不适合替代什么

- 最终玩法判断
- 最终公平性机制设计
- 最终数值平衡
- 最终美术统一性拍板

## 对当前推荐项目的现实建议

对 `Word Sprint Duel V1`，这套最值得用在：

- UI 文案和 onboarding
- 视觉方向探索
- 宣传材料
- 可选的 announcer / TTS 播报

而不是把核心对战 loop 做成 AI 驱动。
