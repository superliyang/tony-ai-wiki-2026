# Convai

## 它代表什么

`Convai` 代表的是更低门槛、面向创作者和开发者的 AI character 平台路线。

官方强调：

- building, customizing, and deploying intelligent interactive AI characters
- Unity / Unreal / Web plugins
- no-code experiences
- API reference

## 为什么值得放进知识库

因为它说明：

- AI NPC 已经不是纯研究玩具
- 角色配置、引擎插件、web 体验、modding，都在产品化

## 对 skills gaming 的判断

如果你们做的是：

- 更 narrative 的休闲竞技壳子
- 更需要 host / guide / tutor
- 更重社交体验的外围层

它有参考价值。

如果你们做的是极简 skills duel，优先级就没那么高。
