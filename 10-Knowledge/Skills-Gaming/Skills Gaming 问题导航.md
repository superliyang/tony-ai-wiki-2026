## 如果你不知道自己属于哪类问题

- 偏赛道判断：先看 `市场 / 商业模型 / owner 视角`
- 偏全局理解：先看 `休闲竞技全景 / 休闲与竞技 / 竞技类型 / 开发步骤`
- 偏玩法判断：先看 `玩法原型 / game feel / pacing`
- 偏系统可信度：先看 `fairness / score validation / anti-cheat`
- 偏经营与增长：先看 `留存 / UA / LiveOps / support`
- 偏项目落地：先看 `作战手册 / 项目总览 / 首轮试玩计划`
