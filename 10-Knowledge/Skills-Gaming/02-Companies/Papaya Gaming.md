# Papaya Gaming

## 定位

从官网表达看，`Papaya` 更像以自营 portfolio 为核心的 skill-based gaming 公司。

## 关键特征

- 强调 own games portfolio
- 强调 skill-based matchmaking
- 强调高频 tournaments
- 强调活动、社交和持续内容更新

## 对你的启发

它提醒我们：

- 一款成功的 skills game 不只是“能赢钱”
- 它还要持续给玩家：
  - 熟悉的 loop
  - 清晰的 skill ceiling
  - 高频回流理由
  - 活动和社交化的留存点
