# Knowledge Representation

## 为什么重要

- 想让机器“推理”，首先得解决“知识怎样表示”
- 这是 symbolic AI 的核心问题，也影响今天的知识图谱、RAG 和工具系统

## 核心问题

- 事实如何表示
- 关系如何组织
- 不确定信息如何处理

## 常见表示方式

- 规则
- 逻辑表达
- 图结构
- 向量表示

## 和今天的关系

- 现代表示方式已经从显式符号扩展到 embedding 和 latent representation
- 所以“知识表示”今天依然存在，只是形式变了

## 今天它以什么形式活着

- RAG
- structured context
- memory
- tool schema
- embedding / latent representation

## 今天最容易误解它的地方

- 不是“现在没有 knowledge representation 了”
- 而是知识从显式结构扩展到了隐式表示和混合系统
