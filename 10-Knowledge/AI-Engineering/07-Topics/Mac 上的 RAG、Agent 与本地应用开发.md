# Mac 上的 RAG、Agent 与本地应用开发

## 为什么这页重要

因为真正让你变强的，不只是会跑模型，而是会把模型接成系统。

## 推荐的本地项目顺序

### 1. Local Chat

- 本地模型
- 简单 UI
- 基本 prompt tuning

### 2. Local RAG

- 文档切分
- embedding
- retrieval
- answer grounding

### 3. Local Eval

- 构建测试集
- 比较 prompt / model / retrieval 变化

### 4. Local Agent

- tool calling
- file / shell / browser 小工具
- approval boundary

## 这条线和专家路径的关系

- `Foundations` 让你知道原理
- `Learning` 让你知道模型、系统、生态
- `Engineering` 让你会把它们做出来
- 本地 Mac lab 则让你反复练熟
