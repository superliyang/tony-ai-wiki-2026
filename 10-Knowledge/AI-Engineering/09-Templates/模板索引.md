# 模板索引

这一层放 AI Engineering 可复用模板。模板要服务于重复工作：项目 intake、实验记录、评测设计、上线检查、复盘和知识提升。

## Template Buckets

| Template | 用途 | 状态 |
|---|---|---|
| Project Intake | 判断一个工程项目是否值得进入正式域 | 待提升 |
| Experiment Log | 记录假设、配置、结果、失败原因 | 待提升 |
| Eval Design | 设计数据集、rubric、回放和 gate | 待提升 |
| Release Checklist | 上线前检查 model、tool、permission、fallback | 待提升 |
| Incident Review | 记录 AI 系统故障、根因和修复动作 | 待提升 |
| Promotion Note | 从旧库或 working vault 提升正式笔记 | 待提升 |
