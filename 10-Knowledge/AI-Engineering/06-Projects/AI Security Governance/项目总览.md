# AI Security Governance

## 项目定位

这是一个围绕企业 AI / agent 系统的安全治理、评审与上线机制的文档包。

它的目标不是实现某个单一安全产品，而是先把这些问题沉淀清楚：

- 企业该如何组织 AI security review
- release gate 怎么设计才不会流于形式
- incident / exception / telemetry 该由谁负责
- 如何把 tool、memory、gateway、artifact 安全连成一条线
