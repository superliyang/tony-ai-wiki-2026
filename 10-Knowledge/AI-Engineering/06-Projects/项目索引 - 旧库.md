## 怎么使用这一层

- `Topics` 讲抽象原理
- `Maps` 讲结构关系
- `Projects` 讲“如果我们真要做一个系统，需求文档和设计文档该怎么写”
- `Harness Lab` 偏 harness 与动作面实践
- `AI Coding Lab` 偏 coding agent 的个人能力训练、团队试点、权限治理、任务记录和验收矩阵
- `Inference Capacity Lab` 偏成本、服务层级与容量规划实践
- `Agent Runtime Lab` 偏 session、memory、background execution、delegation、postmortem 和 rollout gate 实践
- `Memory Lab` 偏 state / memory / artifact 分层、写入策略、后台 consolidation、记忆评测，以及从 learnings 到 skill extraction、incident-fed eval、plugin wrapper、live cache / hook payload 调试、incident pack replay 和 candidate-skill rollout gate 的实践
