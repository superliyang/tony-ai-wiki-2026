# 项目总览

## 这个项目包是干什么的

`Agent Runtime Lab` 不是让你立刻造一个完整 agent 平台，而是把最容易说空的几层：

- session
- memory
- background execution
- delegation
- review handoff
- postmortem
- rollout gate

压成可执行的学习实验。
