## 评估维度

| 维度 | 要问的问题 | 判断 |
|---|---|---|
| 规模 | 目标人群是否足够大？是否有扩量空间？ |  |
| 成本 | CPM/CPC/CPI/CPA 与现有渠道相比如何？ |  |
| 质量 | 留存、付费、退款、LTV 是否可接受？ |  |
| 增量 | 是新增用户，还是抢自然量/其他渠道 credit？ |  |
| 透明度 | 能否看到媒体、版位、地区、素材和事件数据？ |  |
| 风险 | 是否有 fraud、brand safety、合规风险？ |  |
| 运营 | 是否需要代理商、人力、素材或工程支持？ |  |
