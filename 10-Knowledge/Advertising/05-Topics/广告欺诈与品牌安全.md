# 广告欺诈与品牌安全

## 核心观点

便宜流量不是便宜获客。广告欺诈、低质媒体、错误归因和品牌安全问题，会把“看起来很好的投放报表”变成真实经营损失。

## 常见欺诈与低质形态

- Bot traffic：机器人曝光、点击或转化。
- Click spam：伪造大量点击，等待自然转化被归因。
- Click injection：在安装前瞬间注入点击，抢归因。
- SDK spoofing：伪造 SDK 事件。
- Install farm：人工或设备农场制造安装。
- Domain spoofing：伪装成高质量媒体域名。
- Ad stacking：多个广告堆叠，用户实际只能看到一个。
- Pixel stuffing：把广告塞进极小像素区域。

## 防御分层

1. 投前：渠道准入、白名单、黑名单、PMP、SPO、合同条款。
2. 投中：异常 CTR/CVR、留存、付费、设备、地区、媒体、时间分布监控。
3. 投后：MMP fraud report、BI cohort、退款、拒付、LTV、渠道结算复核。
4. 商务：对账、扣款、返点、赔付、媒体复审。
5. 组织：把 fraud review 纳入预算例会，而不是事故后补救。

## 判断信号

- CPI 极低但留存极差。
- 点击到安装时间分布异常集中。
- 某些媒体或子渠道占比异常。
- 付费率、退款率、设备型号、地区分布和正常用户差异巨大。
- 平台报表很好，但 BI 收入和留存没有响应。
