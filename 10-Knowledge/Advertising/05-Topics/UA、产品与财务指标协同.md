# UA、产品与财务指标协同

## 核心观点

广告投放不是 UA 团队单独完成的动作。能不能持续放量，取决于 UA 买来的用户、产品承接、变现效率、财务约束和管理目标是否在同一套系统里。

## 常见冲突

- UA：希望扩大预算，追求平台 ROAS 或 CPI。
- 产品：希望优化激活、留存、付费转化。
- 数据：希望统一事件、口径、归因和实验设计。
- 财务：关心净收入、毛利、现金回收和预算风险。
- 管理层：关心增长速度、市场份额、利润和长期壁垒。

## 对齐方式

1. 统一用户旅程：曝光 -> 点击 -> 安装 -> 激活 -> 留存 -> 付费 -> 复购 -> 回本。
2. 统一指标字典：每个指标要有公式、数据源、Owner 和使用场景。
3. 统一预算机制：预算不是按感觉加，而是按边际效率、现金约束和战略阶段加。
4. 统一实验节奏：素材、渠道、产品漏斗和价格实验不要互相污染。
5. 统一复盘机制：每周看执行，每月看结构，每季度看增量和战略。

## 专家问题感

- 这个渠道的增长是真增量，还是抢了自然量和其他渠道的 credit？
- 当前瓶颈是在媒体采买、素材供给、产品转化、定价，还是回本周期？
- 如果把奖金从平台 ROAS 改成 D30 Net ROAS，会改变团队哪些行为？
- 如果现金 payback 从 30 天放宽到 60 天，增长曲线和风险敞口会怎么变？
