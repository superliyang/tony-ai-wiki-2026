## 2026-06 Refresh Lens

广告投放近三个月最值得跟的不是单条平台新闻，而是三类结构性变化：

| Direction | 变化 | 对知识库的影响 |
|---|---|---|
| Platform API Versioning | Google Ads API、DV360 API、Meta Graph / Marketing API 持续变更 | 投放工具、数据拉取、权限和报表口径要跟 API 版本走 |
| AI Ads Workflow | Google Marketing Live 2026 强调 AI-assisted creative、campaign building、demand capture | 广告知识图谱要补 creative automation 和 AI-assisted media buying |
| Measurement / Data Governance | 平台 API、数据保留、归因和 privacy 变化影响长期报表 | 归因和投放复盘不能只依赖旧指标口径 |
| MCP / Automation Interfaces | TikTok Ads MCP 等信号出现 | 广告平台开始进入 agent/API automation 视角，需要和 AI Engineering / Security 联动 |
