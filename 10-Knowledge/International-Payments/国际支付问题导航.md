## 如果你不知道自己属于哪类问题

- 偏收入和转化：先看 `支付成功率 / 支付方式 / 订阅恢复`
- 偏资损和风险：先看 `拒付 / 争议 / 风控 / 认证`
- 偏系统与稳定性：先看 `系统架构 / 账本 / Webhook / 发布治理`
- 偏地区扩张：先看 `市场案例 / 重点国家 / 本地方式`
- 偏管理和 owner 视角：先看 `指标看板 / 六层 owner 能力 / 场景作战 / 动作库`
