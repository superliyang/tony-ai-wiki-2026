# OWASP ZAP

## 它是什么

OWASP ZAP 是开源 Web 应用安全测试工具，覆盖代理、主动扫描、被动扫描和自动化测试。

## 为什么现在值得关注

- 适合学习 Web/API 安全测试的完整流程。
- 能进入 CI/CD 做 baseline scan，但更适合授权测试和安全评审辅助。

## 它在 stack 的哪一层

- 更像 `子系统`：Web/API 安全测试工具。

## 最适合它的场景

- Web 安全测试学习。
- 应用上线前安全评审辅助。
- API baseline scan。

## 风险与边界

- 主动扫描要限定环境和速率。
- 不替代人工业务逻辑评审。
