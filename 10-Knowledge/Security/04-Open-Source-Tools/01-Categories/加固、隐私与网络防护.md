# 加固、隐私与网络防护

## 解决什么问题

- 主机基线、WAF、VPN/零信任访问、阻断恶意来源、个人/团队隐私安全。

## 代表项目

- [Lynis](https://github.com/CISOfy/lynis)：Linux/macOS/Unix 安全审计和加固。
- [fail2ban](https://github.com/fail2ban/fail2ban)：认证失败封禁。
- [SafeLine](https://github.com/chaitin/SafeLine)：自托管 WAF。
- [BunkerWeb](https://github.com/bunkerity/bunkerweb)：云原生 WAF。
- [Algo VPN](https://github.com/trailofbits/algo)：个人 VPN 自动化部署。
- [Firezone](https://github.com/firezone/firezone)：基于 WireGuard 的零信任访问平台。
- [StevenBlack hosts](https://github.com/StevenBlack/hosts)：hosts 阻断列表。

## 典型组合

`Lynis -> 基线修复 -> WAF / fail2ban -> VPN / ZTNA -> 日志检测`

## 风险与边界

- WAF 和封禁策略容易误伤，需要灰度和回滚。
- VPN/ZTNA 不等于零信任全部，还要绑定身份、设备态势、应用访问和审计。
