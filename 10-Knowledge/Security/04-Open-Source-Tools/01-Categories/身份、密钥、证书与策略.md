# 身份、密钥、证书与策略

## 解决什么问题

- 管理 human identity、machine identity、secret、证书、访问策略和 policy-as-code。
- 把身份安全从“账号系统”扩展到服务间访问、证书自动化和策略决策。

## 代表项目

- [Vault](https://github.com/hashicorp/vault)：secret 管理、加密服务和访问控制。
- [OpenBao](https://github.com/openbao/openbao)：Vault 生态的开源分支方向。
- [Infisical](https://github.com/Infisical/infisical)：secret、证书和 PAM 平台。
- [Authelia](https://github.com/authelia/authelia)：SSO/MFA 门户。
- [Open Policy Agent](https://github.com/open-policy-agent/opa)：通用策略引擎。
- [cert-manager](https://github.com/cert-manager/cert-manager)：Kubernetes 证书自动化。
- [step-ca](https://github.com/smallstep/certificates)：私有 CA 和 ACME server。
- [Ory Kratos](https://github.com/ory/kratos)：云原生身份管理。
- [Ory Hydra](https://github.com/ory/hydra)：OAuth2/OIDC server。

## 典型组合

`IdP / SSO -> Vault / OpenBao -> cert-manager / step-ca -> OPA -> 审计日志`

## 学习价值

- 理解零信任的底层对象：身份、凭据、证书、策略、上下文和审计。
