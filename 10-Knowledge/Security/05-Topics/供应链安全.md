# 供应链安全

## 定位

供应链安全关注代码、依赖、构建、镜像、发布、第三方 SDK、外包服务和 SaaS 集成是否可信。

现代攻击越来越多不是直接打生产系统，而是污染生产系统依赖的上游。

## 关键问题

- 依赖包是否可信
- 构建环境是否可被篡改
- 镜像和 artifact 是否可验证来源
- CI/CD secret 是否泄露
- 第三方 SDK 是否收集或泄露数据
- 供应商安全能力是否可审计

## 主要控制

- SCA
- SBOM
- artifact signing
- provenance
- dependency pinning
- CI/CD hardening
- secret scanning
- vendor risk review
- secure build pipeline
