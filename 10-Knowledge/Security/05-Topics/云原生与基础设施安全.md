# 云原生与基础设施安全

## 定位

云原生与基础设施安全关注云账号、主机、容器、Kubernetes、网络、存储、CI/CD 和运行时工作负载。

它和传统网络安全不同：边界不再只是一台防火墙，而是身份、配置、工作负载、服务网格和自动化发布链路。

## 关键问题

- 云账号和 IAM 是否最小权限
- 安全组、VPC、存储桶、数据库是否错误暴露
- 容器镜像是否有漏洞或恶意内容
- Kubernetes RBAC、network policy、secret 是否安全
- IaC 配置是否经过扫描和审查
- 运行时异常如何发现

## 主要控制

- CSPM
- CWPP
- CNAPP
- container image scanning
- Kubernetes admission control
- runtime security
- IaC scanning
- cloud IAM review
- workload identity
- secrets management
