# 身份与访问安全

## 定位

身份与访问安全是安全领域的中枢。

很多安全事故表面是漏洞、数据泄露或权限滥用，本质上都是：

- 谁被识别成了谁
- 谁被允许访问什么
- 谁拿到了不该拿的权限
- 权限变化有没有被审计

## 关键问题

- 用户、员工、服务、机器和第三方如何被认证
- 权限如何最小化
- 高权限账号如何管理
- 访问是否需要上下文判断：设备、地点、风险、行为
- secret、API key、证书如何轮换和保护

## 主要控制

- IAM
- SSO
- MFA
- OAuth / OIDC
- RBAC / ABAC
- PAM
- conditional access
- secret management
- workload identity
- access review
