# 经济周期与宏观变量

> 宏观不是背概念，而是理解：钱的价格、风险偏好、需求强弱和政策约束如何影响行业、公司和个人选择。

## 一句话理解

经济周期是信贷、需求、供给、价格、政策和预期的循环变化。

对个人和企业来说，宏观周期最重要的影响是：

- 资金贵不贵。
- 需求强不强。
- 风险偏好高不高。
- 资产价格是否透支预期。
- 企业愿不愿意扩张和招聘。

## 关键变量

| 变量 | 观察意义 | 常见影响 |
|---|---|---|
| 利率 | 钱的价格 | 影响投资、资产估值、企业融资 |
| 通胀 | 购买力和成本压力 | 影响消费、利润率、政策 |
| 就业 | 居民收入和需求 | 影响消费、信心、政治压力 |
| 信贷 | 扩张能力 | 影响地产、企业投资、风险资产 |
| 汇率 | 外部竞争力和资本流动 | 影响出口、进口成本、跨境业务 |
| 财政政策 | 政府支出和转移支付 | 影响基建、产业扶持、需求 |
| 货币政策 | 流动性和融资环境 | 影响资产价格和风险偏好 |
| 人口结构 | 长期需求和劳动力 | 影响消费、养老、地产、教育 |
| 地缘与贸易 | 供应链和市场边界 | 影响制造、能源、科技、跨境 |

## 宏观周期的常见阶段

| 阶段 | 特征 | 企业策略 | 个人策略 |
|---|---|---|---|
| 复苏 | 利率较低、需求修复 | 找增长、补库存、扩渠道 | 增加学习和机会探索 |
| 过热 | 需求强、通胀压力 | 控成本、防过度扩张 | 避免追高和过度杠杆 |
| 滞胀 | 成本高、增长弱 | 提效率、守现金流 | 提高抗风险能力 |
| 衰退 | 需求弱、风险偏好低 | 降本、聚焦核心 | 保现金流、补能力 |
| 再平衡 | 政策转向、结构调整 | 找新利润池 | 重新配置技能和方向 |

## 如何读宏观信息

不要只看单一数字，要看组合：

- 高增长 + 低通胀：扩张环境较好。
- 高增长 + 高通胀：可能接近过热或供给约束。
- 低增长 + 高通胀：企业和居民都难受。
- 低增长 + 低通胀：政策可能转向刺激，但需求弱。

## 宏观到行业的传导

| 宏观变化 | 可能影响 |
|---|---|
| 利率上升 | 高估值科技、地产、长周期资产承压 |
| 利率下降 | 风险资产和融资敏感行业改善 |
| 通胀上升 | 成本转嫁能力强的公司更有优势 |
| 消费走弱 | 可选消费、广告、游戏、教育等承压 |
| 地缘风险上升 | 供应链、本土替代、能源安全变重要 |
| 人口老龄化 | 医疗、养老、自动化、效率工具重要 |

## 用宏观指导个人

宏观不应该让人陷入焦虑，而是帮助判断：

- 职业机会是在扩张行业还是收缩行业？
- 现在更应该追求增长，还是提高安全边际？
- 哪些技能更抗周期？
- 哪些趋势不受短期宏观影响，反而会被周期加速？
