# 宏观认知与趋势判断全景

## 核心判断

宏观认知不是为了预测明天涨跌，而是为了知道自己处在什么时代、什么周期、什么产业迁移中，以及应该如何分配注意力和行动。

趋势判断也不是追风口，而是回答：

- 这个变化是周期性的，还是结构性的？
- 是技术驱动，还是政策驱动，还是资本驱动？
- 是供给变化，还是需求变化？
- 谁受益，谁受损？
- 多久会兑现？
- 我能采取什么行动？

## 五层世界模型

| 层级 | 看什么 | 典型问题 |
|---|---|---|
| 宏观层 | 利率、通胀、就业、财政、货币、人口、贸易 | 当前环境鼓励扩张还是收缩？ |
| 科技层 | 技术成熟度、成本曲线、性能突破、基础设施 | 哪些技术从实验室进入产品化？ |
| 产业层 | 价值链、供需、渠道、监管、利润池 | 变化会落到哪些行业和环节？ |
| 商业层 | 商业模式、竞争格局、组织能力、资本效率 | 谁能把趋势变成利润？ |
| 个人层 | 职业、技能、资产、创业、关系网络 | 我该怎样调整行动？ |

## 趋势判断的四个时间尺度

| 时间尺度 | 关注点 | 适合的行动 |
|---|---|---|
| 1 周-1 月 | 新闻、情绪、事件冲击 | 记录，不轻易下结论 |
| 3-12 月 | 政策、资本、产品节奏 | 小规模试探、学习、观察指标 |
| 1-3 年 | 产业迁移、商业化、组织变化 | 技能投入、项目选择、职业转向 |
| 3-10 年 | 技术范式、人口、能源、地缘结构 | 长期战略、资产配置、人生方向 |

## 判断趋势是否重要

一个趋势值得进入知识库，至少满足其中 3 条：

- 影响多个行业。
- 改变成本结构。
- 改变用户行为。
- 改变组织方式。
- 带来新基础设施。
- 有政策、资本或技术多重推动。
- 已经出现真实商业化案例。
- 能影响个人职业和技能策略。

## 防止误判的四个问题

1. 我看到的是事实，还是观点？
2. 这个观点的反方是什么？
3. 这个趋势是否已经被价格和预期充分反映？
4. 如果我错了，最可能错在哪里？

## 从趋势到行动

| 趋势判断 | 可行动问题 |
|---|---|
| 技术正在成熟 | 我该学习什么能力？ |
| 行业利润池迁移 | 哪些公司或岗位会变得重要？ |
| 宏观风险上升 | 我是否该提高现金流和抗风险能力？ |
| 平台规则变化 | 产品和增长策略是否要换？ |
| 监管趋严 | 哪些商业模式会受限，哪些合规能力变值钱？ |
