# 模板索引

## Template Buckets

| Template | 用途 |
|---|---|
| Company Card | 记录组织定位、模型、产品、关键人物 |
| Model Card | 记录发布时间、能力、限制、来源 |
| Paper Note | 记录问题、方法、结论、影响 |
| Timeline Event | 记录 dated event 和后续影响 |
| Topic Note | 从多来源提炼稳定主题 |
