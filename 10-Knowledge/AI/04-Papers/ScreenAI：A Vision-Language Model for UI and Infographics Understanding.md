# ScreenAI：A Vision-Language Model for UI and Infographics Understanding

## 简介

`ScreenAI` 是最近两年特别值得补的一张感知层论文卡，因为它不是泛泛地谈视觉理解，而是直指 `UI` 和 `infographic` 这种 agent 真正会碰到的界面。

## 核心想法

它把界面理解、屏幕元素、信息图和复杂视觉布局当成一类专门能力来建模，而不是把这些都混进普通图像理解里。

## 为什么重要

- 它说明 `computer use` 的前提不是先会点击，而是先会理解界面
- 它把 document / screen / infographic 这些视觉工作流重新拉回多模态主线上
- 它也让 UI 感知和 document AI 更容易进入同一套工程思考
