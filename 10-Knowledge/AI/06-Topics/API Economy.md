# API Economy

## 这个主题是什么

`API Economy` 关注模型能力如何通过接口变成平台收入、开发者生态和产品基础设施。

## 为什么重要

过去半年，这个主题已经不再只是“谁家 API 更便宜”，而是：

- 谁能提供更完整的 tool surface
- 谁能提供更好的 agent builder workflow
- 谁能提供更稳定的 app server / session / eval surface
- 谁能更容易进入 enterprise / sovereign deployment

## 最近半年最值得注意的变化

- [[OpenAI]]：从 API provider 更像平台 provider，`AgentKit`、`Codex harness` 说明接口已经开始向工作台升级
- [[Anthropic]]：`Claude Agent SDK` 和 computer use 让 API 叙事更贴近 agent execution
- [[xAI]]：`Agent Tools API` 说明 API 层也开始内建 tool use surface
- [[DeepSeek]]：继续用成本优势和能力更新拉动 API adoption
- [[Moonshot AI]]、[[MiniMax]]、[[Zhipu AI]] 也都在往 OpenAI-compatible + agent-friendly 的开发者入口靠拢
