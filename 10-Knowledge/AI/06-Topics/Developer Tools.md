# Developer Tools

## 这个主题是什么

`Developer Tools` 关注模型公司如何通过 API、SDK、CLI、IDE integration 和 workflow tools 进入开发者生态。

## 为什么重要

最近半年，一个非常明显的变化是：frontier 公司不再只卖模型，而是在卖开发者工作流。

## 最近半年最值得注意的变化

- [[OpenAI]]：`AgentKit`、`Codex GA`、`Codex harness / App Server`
- [[Anthropic]]：`Claude Agent SDK`、`Xcode integration`
- [[Google DeepMind]]：`Gemini CLI`、`Google Antigravity`
- [[Alibaba Cloud]]：`Qwen Code`
- [[Moonshot AI]]：`Kimi CLI`
- [[xAI]]：`Agent Tools API`

这意味着 developer tools 已经从“模型周边生态”升级成 frontier 竞争主线之一。
