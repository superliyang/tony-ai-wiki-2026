# Gemini 3 Deep Think

## 简介

`Gemini 3 Deep Think` 是 Google DeepMind 在 `2026-02-12` 更新的 specialized reasoning mode，也是这轮最值得单独记住的 reasoning 路线节点之一。

## 为什么重要

- Google 直接把它定位成面向 `science / research / engineering` 的 specialized reasoning mode
- 它不只是一个内部实验，而是已经进入 Gemini app，并对研究者、工程师和企业开放 API 早期访问
- 它说明 Google 正在把“通用模型”和“专门推理模式”做成更清楚的双轨路线

## 这条路线代表什么

- `Gemini 3 Deep Think` 代表的是 reasoning specialization
- 它更像是为 hardest problems 准备的推理层，而不是所有人日常默认使用的聊天层

## 学习时重点看什么

- 为什么 specialized reasoning mode 会在 `2026` 重新变得更重要
- Google 为什么强调 science、research、engineering 这些高复杂度场景
- 它和 [[Gemini 3-1 Pro|Gemini 3.1 Pro]] 的关系：一个更像前沿 reasoning 尖刀，一个更像大规模分发的主力层

## 你学习这个模型时最该建立的判断

- reasoning 不再只是“每个模型都更会想一点”，而是在被做成独立层次
- 这条线很适合拿来理解为什么未来的 frontier 竞争可能不再只围绕一个默认聊天模型展开
