# GPT-5.4

## 简介

`GPT-5.4` 是 OpenAI 在 `2026-03-05` 发布的新旗舰节点，也是这轮最该记住的通用模型升级之一。

## 为什么重要

- 它是 OpenAI 首个带 `native computer use` 的通用模型
- 它把 `1M context`、tool search、computer use 和 coding 进一步收成一个统一旗舰面
- 它说明 OpenAI 正在把“通用最强模型”往 `long-horizon professional agent` 方向推进，而不只是继续做聊天模型

## 这条路线代表什么

- `GPT-5.4` 代表的是 `general-purpose model -> long-horizon agent backbone`
- 它不只是更会回答，而是更像能跨工具、跨应用、跨长任务执行与验证的通用工作模型

## 学习时重点看什么

- 为什么 `native computer use` 是一个真正的系统跃迁，而不只是 demo feature
- `1M context` 和 tool search 如何改变 agent 的工作集大小与工具组织方式
- 它和 [[GPT-5-3-Codex|GPT-5.3-Codex]] 的关系：一个更像 coding flagship，一个更像通用 long-horizon flagship

## 你学习这个模型时最该建立的判断

- OpenAI 的旗舰路线已经不再只是“更好的 reasoning”，而是“更好的 reasoning + 更好的工具生态工作能力”
- `GPT-5.4` 很适合拿来理解为什么今天的 frontier 模型竞争越来越像在争夺 `computer use + tool ecosystems + long tasks`
