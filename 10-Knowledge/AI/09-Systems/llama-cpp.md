# llama.cpp

## 它是什么

`llama.cpp` 是本地 LLM runtime 的经典代表之一。

## 为什么它适合专家路径

因为它能逼你真正理解：

- quantization
- KV cache
- context length
- runtime tradeoffs
- CPU / GPU / Metal backend

## 它最适合什么

- 理解本地推理 runtime
- 比较不同模型和量化形式
- 做更底层的性能实验

## 对 Mac 用户的意义

在 Apple silicon 上，它是学习本地推理细节的好入口之一。
