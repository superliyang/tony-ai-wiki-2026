# DeepSeek API

## 简介

`DeepSeek API` 是 DeepSeek 面向开发者和平台生态的接口层。

## 为什么重要

- 它决定了 DeepSeek 如何进入开发者工作流
- 它让模型能力从研究或演示走向可集成的基础设施
- 它有助于观察中国模型公司如何建设平台生态

## 学习时重点看什么

- 它与 [[DeepSeek-V3]]、[[DeepSeek-R1]] 的关系
- 开发者为何会选择这一平台
- 它如何影响 agent 和自动化工作流设计
