# MLX

## 它是什么

`MLX` 是 Apple silicon 取向非常明确的 machine learning framework。

它最值得注意的不是“又一个框架”，而是：

- array framework
- machine learning framework
- designed for Apple silicon
- unified memory aware

## 为什么它重要

对 `Mac M4 Max` 用户来说，`MLX` 是本地学习大模型最有代表性的原生路线之一。

## 它最适合什么

- 本地推理
- LoRA / fine-tuning 实验
- Apple silicon 原生优化
- 把本地 AI 实验做得更轻更顺

## 它不替代什么

- 不替代 `PyTorch` 的生态规模
- 不替代云端分布式训练栈
- 不自动替代所有 inference runtime
