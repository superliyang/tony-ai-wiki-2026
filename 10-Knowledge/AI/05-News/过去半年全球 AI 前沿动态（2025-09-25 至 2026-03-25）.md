# 过去半年全球 AI 前沿动态（2025-09-25 至 2026-03-25）

> 这份整理覆盖 `2025-09-25` 到 `2026-03-25` 之间，全球 AI 生态里对前沿竞争格局最有影响的一批官方动态。它不追求收录每一条融资、每一次小功能更新，而是优先抓：模型发布、agent / coding 平台更新、协议与工作台变化、多模态与 OCR 进展、以及主权化 / 企业化落地。

## 先说结论

过去半年，全球 AI 前沿最明显地收敛成了 6 条大线：

1. `frontier 模型继续快速迭代`：[[OpenAI]]、[[Anthropic]]、[[Google DeepMind]]、[[xAI]]、[[DeepSeek]]、[[MiniMax]]、[[Zhipu AI]] 都在把 reasoning、coding、tool use、computer use 拉成主能力。
2. `agent / coding 系统化成为主战场`：模型不再只是“答题器”，而是在往 agent runtime、coding workflow、tool API、app server、SDK 方向长。
3. `上下文、工具、协议和 harness 成为平台分水岭`：`AgentKit`、`Claude Agent SDK`、`Codex harness / App Server`、`MCP`、`Agent Tools API` 都说明平台工程正在替代单纯 prompt engineering 成为竞争核心。
4. `中国 frontier 厂商明显往 reasoning + coding + agent 叙事收敛`：[[DeepSeek]]、[[Moonshot AI]]、[[MiniMax]]、[[Zhipu AI]]、[[Baidu]] 的官方更新都在强调思考模式、tool use、agent 任务、编码能力和原生多模态。
5. `开放权重与全球化并没有退场`：[[Mistral AI]] 和 [[Cohere]] 继续分别在 open-weight、多语言、文档理解、主权 AI 上占据独特位置。
6. `主权化 / enterprise AI 从概念走向系统化落地`：从 [[xAI]] 的 Enterprise，到 [[Cohere]] 的 sovereign AI，再到 Anthropic 与英国政府、百度千帆、Google enterprise surfaces，这条线已经非常清楚。

## 过去半年最重要的结构变化

### 1. Prompt 不是主角了，workbench 才是主角

这半年最重要的变化，不是又多了几个更强模型，而是 frontier 公司都在把模型包进工作台：

- [[OpenAI]]：从 `Codex GA`、`GPT-5.2-Codex`、`GPT-5.3-Codex` 到 `Codex harness / App Server`
- [[Anthropic]]：从 `Claude Sonnet 4.5` 到 `Claude Agent SDK` 再到 `Xcode` 原生接入
- [[Google DeepMind]]：从 `Gemini 3` 到 `Google Antigravity`
- [[xAI]]：从 `Grok 4.1 Fast` 到 `Agent Tools API`
- [[Alibaba Cloud]]：虽然这半年模型级大新闻偏少，但 `Qwen Code` 已明显往 coding agent 方向走

这说明 frontier competition 已经从“谁的模型分高”变成“谁能把模型变成稳定工作流”。

### 2. Coding / Agent 是闭源 frontier 的共同主线

在过去半年里，最一致的一条产品叙事是：

- coding 不再是一个 benchmark 分支，而是 agent 的核心试验场
- terminal、browser、tool use、memory、approval、session、app server 都在围绕 coding agent 先成熟
- frontier 公司开始把“工程环境”作为模型能力的一部分来设计

这也是为什么我们在 [[../07-Maps/Agent Prompt-Context-Harness Map|Agent Prompt-Context-Harness Map]] 和 [[../../AI-Engineering/07-Topics/Harness Engineering|Harness Engineering]] 里把 `harness` 提到了上位层。

### 3. 中国厂商的叙事正在从“追 benchmark”转向“做 agent-ready 模型”

过去半年里，这个趋势特别明显：

- [[DeepSeek]]：`V3.2` 直接强调 reasoning-first、thinking in tool-use、agent training data synthesis
- [[Moonshot AI]]：`Kimi K2 Thinking`、`K2 Vendor Verifier`、`CLI support` 都在围绕 agent/coding 落地
- [[MiniMax]]：`M2.5` 直接把 coding、agent、office work 摆到最前面
- [[Zhipu AI]]：`GLM-5`、`GLM-4.7-Flash`、`GLM-4.6V` 都明显面向长程 agent、编码和原生工具调用
- [[Baidu]]：`文心 5.0` 则把原生全模态统一建模拉成差异化路线

### 4. “全球化”不再只是美中双强，欧洲和加拿大也有清晰位置

- [[Mistral AI]]：继续把 open-weight、多语言、多模态和 OCR 拉在一起
- [[Cohere]]：继续守住 multilingual + sovereign AI 这条独特路线

这意味着“全球前沿动态”不能只看 OpenAI / Anthropic / Google，也要看 open-weight、文档智能、多语言和主权化这几条非硅谷专属路线。

## 这一轮全局上最该记住的 5 个判断

1. `模型竞争正在快速平台化`：不是只看哪个模型分高，而是看它是否拥有 SDK、tool surface、session、approval、app server、evals。
2. `agent 能力已经是基础能力，不是附加能力`：tool use、computer use、tool-call reasoning、coding、memory 正在变成主能力。
3. `中国公司更强调 reasoning + tool use + agent performance`，而不是单纯“开源 / 闭源”的标签之争。
4. `开放权重路线仍有全球影响力`，尤其在 multilingual、OCR、文档智能和 edge / enterprise deployability 上。
5. `enterprise / sovereign AI` 已经不是“未来趋势”，而是最近半年里官方新闻流里非常高频、非常主流的方向。
