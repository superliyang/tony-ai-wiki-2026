## 当前规范

### 文档成熟度

用于判断主题页、系统页、开源项目页、模板页、地图页、索引页和项目页是否足够可学习、可决策和可落地。

日常补强优先级：

1. 入口页和恢复页
2. 索引和地图
3. 高复用模板
4. 当前主线文档
5. 长尾资料页

详细标准看：[[文档成熟度补强标准]]

### 图谱资产

用于管理 ChatGPT 生成的全景图、架构图、图片、Mermaid、Excalidraw、Canvas、SVG 和 prompt。

推荐流程：

1. 图放到对应领域 `Maps/_assets/images/`
2. 源文件放到 `Maps/_assets/sources/`
3. 提示词放到 `Maps/_assets/prompts/`
4. 建 Markdown 说明页
5. 更新领域 `地图索引.md`
6. 需要时更新 `专题总览.md`

日常操作看：[[./Diagram-Assets/怎么使用图谱资产管理规范|怎么使用图谱资产管理规范]]

生成新图时优先复制：[[./Diagram-Assets/图谱提示词库：全景图、明细图与资产归档|图谱提示词库]]
