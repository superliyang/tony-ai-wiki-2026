# Payments Advanced Layers

## Purpose

Use these layers when the user no longer only needs a solid payments foundation, but wants the knowledge required to operate like a senior payments owner.

## The Six Layers

1. Payment methods
- 卡支付全景
- 钱包支付全景
- A2A 与银行转账支付
- Direct Debit 与代扣支付
- BNPL 与分期支付
- 本地支付方式方法论

2. Compliance operations
- KYC 与 KYB 实战
- AML 与制裁筛查流程
- 商户准入与高风险行业治理
- PCI 审计材料与范围收缩
- 不同地区的数据合规差异

3. Treasury and funds management
- 预充值与 Prefunding 管理
- Rolling Reserve 与 Reserve Policy
- Settlement Calendar 与结算节奏管理
- 多币种资金池管理
- FX 风险与对冲
- 多实体收款与结算架构

4. Data governance
- 指标字典与口径治理
- 埋点与支付事件模型
- 支付实验设计与评估方法
- 告警阈值配置台账
- 仪表盘治理规则

5. Management governance
- 月度支付治理清单
- 风险例会清单
- 通道季度复审清单
- Vendor 与 PSP 管理机制
- On-call 与协同升级机制

6. Commercial and channel procurement
- 商务与通道采购层索引
- PSP 与 Acquirer 选型框架
- 费率谈判与商务条款
- SLA、赔付与服务等级条款
- Liability Allocation 与责任划分
- Reserve、Settlement 与商务约束条款
