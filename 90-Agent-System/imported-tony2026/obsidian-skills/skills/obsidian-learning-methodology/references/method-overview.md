# Method Overview

## Goal

Turn a topic into a durable learning system instead of a pile of notes.

## Core Loop

1. scope the domain
2. choose the layers
3. create indexes and maps
4. add notes in graph-aware order
5. update progress and resume memory
6. checkpoint meaningful changes

## Success Criteria

A domain is in good shape when:

- new notes have an obvious home
- reading order is clear
- progress and resume notes are current
- maps explain structure at a glance
- the vault survives interruption well
