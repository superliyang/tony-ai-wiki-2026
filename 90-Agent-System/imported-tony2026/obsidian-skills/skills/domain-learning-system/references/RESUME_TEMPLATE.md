# Resume Template

Use one short resume note per domain when the user may pause and return later.

```markdown
---
title: 恢复笔记
type: resume
status: active
domain:
updated:
---

# 恢复笔记

## Last Session

- We studied:
- We finished:

## What I Actually Understand Now

- 

## What Still Feels Fuzzy

- 

## Restart Here

- Read:
- Then continue to:

## Do Not Re-read Everything

- Skip:
- Review only if stuck:
```

## Usage Notes

- This note is a restart aid, not a full summary.
- Aim for a 2 minute re-entry experience.
- If the user changes domains, update this note before switching.
