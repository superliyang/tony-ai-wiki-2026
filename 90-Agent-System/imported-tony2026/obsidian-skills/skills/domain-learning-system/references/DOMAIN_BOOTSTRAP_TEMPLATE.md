# Domain Bootstrap Template

Use this template when creating a new study domain.

```markdown
# <Domain Name>

## Purpose

- Why this domain matters
- What "knowing enough" looks like

## Major Sections

1. Foundations
2. Landscape
3. Systems / Operations
4. Applications

## Suggested Folder Shape

<Domain Name>/
├── 00-Inbox/
├── 01-Foundations/
├── 02-Landscape/
├── 03-Systems/
├── 04-Applications/
├── 05-Topics/
├── 06-Maps/
├── 07-Templates/
├── 学习进度.md
└── 恢复笔记.md

## First Five Starter Notes

- 专题总览.md
- 主题索引.md
- 学习进度.md
- 恢复笔记.md
- One "big picture" topic note

## Recommended Learning Order

1. Big picture
2. Foundations
3. Core systems
4. Representative cases
5. Current changes and edge cases
```

## Usage Notes

- Rename sections if the domain needs different language, but keep the logic.
- Keep the first version narrow. Expand after the first few study sessions.
