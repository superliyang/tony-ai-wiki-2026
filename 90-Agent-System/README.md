# Agent System

`90-Agent-System/` stores AI capability assets: prompts, rules, workflows, reviews, decisions, skills, scripts, and integrations.

It answers:

```text
How should agents become more reliable over time?
```

## Structure

| Path | Purpose |
|---|---|
| `prompts/` | reusable prompts before they become workflows or skills |
| `rules/` | Codex, Cursor, Claude, Hermes, and tool-specific rules |
| `workflows/` | repeatable operating procedures |
| `reviews/` | evaluations of agent outputs, automations, and rules |
| `decisions/` | durable architecture and boundary decisions |
| `skills/` | reviewed skill designs and skill notes |
| `scripts/` | script notes and automation design records |
| `integrations/` | Hermes, Feishu, Weixin, GitHub, Obsidian integration notes |

## Important Distinction

```text
90-Agent-System/ documents and reviews capability assets.
Root-level skills/ and scripts/ contain executable capability assets.
```
