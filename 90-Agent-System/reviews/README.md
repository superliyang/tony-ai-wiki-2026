# Reviews

Reviews evaluate AI outputs, automations, skills, rules, and memory updates.

Good review notes should capture:

- what was evaluated;
- what worked;
- what failed;
- what should change;
- whether the result becomes a rule, playbook, skill, or discarded pattern.
