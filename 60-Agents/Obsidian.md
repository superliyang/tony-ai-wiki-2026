# Obsidian

## Role

Obsidian is the reading, thinking, editing, and graph navigation surface.

Use Obsidian for:

- reading durable knowledge;
- editing human-facing notes;
- navigating maps and wikilinks;
- reviewing AI-generated candidates;
- shaping domain learning paths.

## Boundary

Obsidian is a workspace, not the only memory system. Markdown plus Git makes its content portable.
