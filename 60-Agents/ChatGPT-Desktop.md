# ChatGPT Desktop

## Role

ChatGPT Desktop is a thinking, source intake, synthesis, and conversation entry point.

Use ChatGPT for:

- exploring ideas with Tony;
- summarizing web sources and documents;
- drafting frameworks and candidate structures;
- helping identify what deserves deeper research.

## Writes

ChatGPT should normally hand content to Tony or Codex for placement. If connected to file workflows later, it should write only to staging first.

## Boundary

ChatGPT conversations are not durable memory unless important decisions are written into Markdown.
