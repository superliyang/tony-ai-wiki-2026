# GitHub

## Role

GitHub is the durability, history, backup, and publication layer.

Use GitHub for:

- preserving meaningful checkpoints;
- reviewing diffs;
- syncing the vault across machines;
- publishing selected knowledge if Tony chooses.

## Boundary

Do not publish secrets, local auth files, runtime app state, or unreviewed private staging material.
