## If You Want To Learn

1. 先读目标领域的 `专题总览.md`。
2. 再进入旧库对应领域的 `专题总览.md`、`学习进度.md`、`恢复笔记.md`。
3. 用 [[30-Playbooks/领域专家感学习法]] 生成领域全貌图和专家问题。
4. 如果内容值得沉淀，用 [[30-Playbooks/知识提升流程]] 提升为正式笔记。

1. 选一个领域或一个问题。
2. 从 [[20-Maps/旧库迁移地图]] 找旧库入口。
3. 按 [[90-Agent-System/workflows/legacy-vault-migration]] 做提升。
4. 更新目标领域总览、总地图或本页。
