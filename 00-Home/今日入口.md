# 今日入口

> [!tip] 先选一个方向
> 打开 Obsidian 后，不从文件树随机找笔记。先选你今天是要读知识、推项目、让 AI 做事，还是在线回顾。
>
> 读知识时，默认从 [[20-Maps/核心知识网络]] 或 [[20-Maps/领域平铺图谱]] 开始。前者回答领域之间的关系，后者按领域平铺入口。

## 今天最可能的路径

```text
今日入口
  -> 核心知识网络 / 领域平铺图谱
  -> 选择一个领域
  -> 学习路径 / 专题总览 / 学习状态
```

如果不是读知识，而是继续建设系统：

```text
今日入口
  -> AI First 项目下一步
  -> Review Gate batch decision
  -> Codex / Hermes 执行
```

## 当前系统分工

```text
Obsidian = 个人知识生产中心
GitHub private repo = 长期版本事实源
output-feishu = 可发布中间层
飞书知识库 = 任意地方访问 / 移动端阅读 / 分享协作
飞书 CLI = 自动同步执行器
```
